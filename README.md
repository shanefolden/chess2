
# Chess 2, new and improved      

&nbsp;
## Installation Instructions 
 
&nbsp;
&nbsp;

Clone the Repository
```
$ git clone https://github.com/shanefolden/chess2.git
```
&nbsp;

Navigate into the directory
```
$ cd chess2
```
&nbsp;

Compile and run the program
```
javac *.java && java Chess
```
